# ProjectSW
Roguelike SongBuilder

Game Objective 
  Defenda os resquícios da humanidade frente às marés da Ausência. Este tower defense transforma música em poder e estratégia. Vença as hordas de inimigos criando suas próprias melodias pensadas para cada batalha.
    Descubra mais sobre a Cantora e o mundo de Olena enquanto enfrenta a Primeira Calamidade. Aumente seu poder como receptáculo de uma divindade antiga e encare o peso de decisões sobre as vidas que estão sob sua guarda. 
    Neste roguelike cada tentativa é um novo concerto e um repertório de possibilidades.
